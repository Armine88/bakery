# Tinkerer based bakery

## Requirements

http://tinkerer.me/

## Installation

To install the requirements for tinkerer, use virtualenv and pip:

```bash
cd /path/to/bakery
virtualenv env
. env/bin/activate
pip install -r requirements.txt
```

## Usage

Clone this repo and run the following command in the project folder:

```bash
tinker --build
```
